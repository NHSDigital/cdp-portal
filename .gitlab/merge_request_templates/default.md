- [ ] Branch name follows correct naming convention (<dev_initials>-sde-<ticket_number>-<purpose_of_ticket>)
- [ ] MR name follows correct naming convention (sde-<ticket_number>-<merge_details>) 
- [ ] Commit messages follow convention (sde-<ticket_number> <purpose_of_the_commit>) 
- [ ] Confirmed deployed and tested on Dev with evidence attached to JIRA ticket.
- [ ] MR pipeline completed successfully
- [ ] Required documentation updates completed e.g. KOPS, README.md   
- [ ] Deployed to a polyfucated environment
- [ ] Comments added regarding incomplete checklist task.

- [ ] Open sourcing of project taken into account
- [ ] Polyfucated Environments Deployed and/or considered
- [ ] Comments added regarding incomplete checklist task.

### Merger Checklist (Before merging the following should be completed.):
- [ ] MR pipeline successful
- [ ] Approval provided after latest code changes
- [ ] Comments added regarding incomplete checklist task.
